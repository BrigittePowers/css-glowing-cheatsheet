CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Now it has a permanent home.
Select any snippet below and it'll automatically select all the code for you to copy.

Flexbox Row

Creates a flexbox row layout from left-to-right.

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout.

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, 1fr);
}

Linear Gradients

Creates background linear gradient top-to-bottom.

.linear-gradient-background {
    background-image: linear-gradient(
        rgba(232, 102, 236, 0.3) 0%,
        rgba(232, 102, 236, 0.6) 100%
    );
}

Box Transition Glow

Use transition and box shadows for a custom hover animation.

.card-header .card-code {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}

.card:hover,
.card:hover
.card-header {
    box-shadow: inset 0px 0px 8px rgba(232, 102, 236, 1), 0 0 15px rgba(232, 102, 236, 1);
}
